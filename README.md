# Personal Site 🌐

This repository contains the source code for my personal website.  
It serves as a central place to showcase my projects, skills, and experience.

## 🚀 Live Site
[View Website](https://nataani.dev)

## 📌 Features
- Responsive design for desktop and mobile
- Project portfolio section
- About / bio section
- Contact information
- Fast static deployment

## 🛠️ Tech Stack
- Typescript
- HTML5
- CSS3
- JavaScript
- Tailwind

## 📂 Project Structure
```
/
├── index.html
├── css/
├── js/
├── assets/
└── README.md
```

Surge allows you to deploy a static site with a single command and host it on a CDN with a custom domain if desired. :contentReference[oaicite:2]{index=2}

## 🧩 Customization

To customize this site:
- Update content in `index.html`
- Modify styles in the `css/` directory
- Add or update scripts in `js/`
- Replace assets (images, icons, etc.)
